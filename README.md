# FitAI - Tu Entrenador Personal con Inteligencia Artificial 🏋️🤖

Una aplicación web completa de fitness personal con inteligencia artificial integrada. Diseñada para transformar tu físico en 12 semanas con rutinas efectivas, periodización real y seguimiento de progreso.

## 🚀 Características

### 💪 Rutinas Efectivas con Periodización Real
- Programa de 12 semanas con fases de adaptación, acumulación, intensificación y pico
- Semanas de deload programadas para recuperación
- Plantillas: PPL, Upper/Lower, Full Body, Bro Split
- Progresión de cargas inteligente

### 🤖 Coach con IA
- Chat interactivo para consultas de entrenamiento
- Generación de rutinas personalizadas
- Análisis de fotos de progreso (con API Key de OpenAI)
- Consejos nutricionales personalizados
- Motivación y guía de recuperación

### 📈 Seguimiento de Progreso
- Registro de entrenamientos con volumen total
- Records personales (PRs) automáticos
- Medidas corporales (peso, grasa, medidas)
- Fotos de progreso
- Gráficas de volumen semanal

### 🥗 Plan Nutricional
- Cálculo automático de macros según objetivo
- Plan de comidas por hora
- Registro de alimentación diaria
- Tracker de agua
- Tips nutricionales personalizados
- Lista de compras semanal

### 📚 Biblioteca de Ejercicios
- 35+ ejercicios con instrucciones detalladas
- Tips de técnica para cada ejercicio
- Filtros por grupo muscular
- Categorización por dificultad

### 👤 Perfil y Valoración
- Valoración física completa (IMC, TDEE)
- Plan de 12 semanas visualizado
- Resultados esperados según nivel
- Exportación de datos

## 🛠️ Tecnología

- **HTML5** + **CSS3** + **JavaScript** puro (sin dependencias)
- **Progressive Web App** compatible
- **LocalStorage** para persistencia de datos
- **OpenAI API** opcional para IA avanzada
- **Diseño mobile-first** oscuro y moderno

## 📱 Uso

1. Abre `index.html` en tu navegador
2. Completa el onboarding (nombre, objetivo, nivel, datos físicos)
3. ¡Empieza a entrenar!

### Para IA avanzada (opcional):
1. Ve a Perfil > Configuración
2. Ingresa tu API Key de OpenAI
3. El chat de IA ahora puede analizar fotos y dar respuestas más inteligentes

## 📊 Plan de 12 Semanas

| Semana | Fase | Intensidad | RPE |
|--------|------|-----------|-----|
| 1-2 | Adaptación | 60-65% | 6-7 |
| 3-4 | Acumulación | 70-72% | 7-8 |
| 5 | **Deload** | 55% | 5-6 |
| 6-8 | Intensificación | 75-80% | 8-9 |

## 🎯 Resultados Esperados

- **Principiante:** 3-6 kg músculo, 30-50% más fuerza
- **Intermedio:** 1.5-3 kg músculo, 10-20% más fuerza
- **Avanzado:** 0.5-1.5 kg músculo, 5-10% más fuerza
